 🍯 Honeypot Dashboard
 🎯 What is this?
A Cowrie SSH Honeypot with a Python Flask dashboard to monitor and analyze attacks in real-time.

 ⚡ Features
- Deployed Cowrie SSH Honeypot on Ubuntu Server
- Captures attacker IPs, passwords, and commands
- Real-time dashboard built with Python Flask
- JSON log analysis

 🛠️ Tech Stack
- Cowrie Honeypot
- Python Flask
- Ubuntu Server 26.04
- Kali Linux (Attack simulation)

 🧠 What I learned
- How attackers perform SSH brute force
- Log analysis and visualization
- Network security monitoring
